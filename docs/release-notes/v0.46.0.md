# v0.46.0 — Local models, first contributors

This release opens a fully local path — Ollama on your own machine, no key
and no cloud — and marks the week WasmHatch stopped being a solo project:
three of the changes below were written by outside contributors.

## Highlights

- **Run on your own machine.** Ollama joins Chrome's built-in model as a
  keyless provider: allow this site with OLLAMA_ORIGINS, pick any model
  you've pulled, and requests never leave your computer. The build-time
  CSP audit still holds — http is allowed for loopback only.
- **One provider picker.** Claude, OpenAI, and OpenRouter behind a single
  key field and dropdown, driven by one registry that also generates the
  CSP allowlist, so the models list and the security policy can't drift
  apart.
- **Download one file at a time.** The file viewer gained a Download
  button with sensible media types — the ZIP backup is no longer the only
  way out. Contributed by @blut-agent (#25).
- **Tool activity in plain words.** All nine tools now describe
  themselves — "Creating a Google Sheet — Q3 budget" instead of
  `create_google_sheet` — via a tested module. Contributed by
  @blut-agent (#26).
- **Key storage, announced.** The API key field and the "Remember on this
  device" checkbox now tell screen readers where the key is stored.
  Contributed by @blut-agent (#23), with an independent matching fix by
  @MFA-G (#24) — thank you both.

## Under the hood

- **Portable agent packages (foundation).** A versioned `wasmhatch.agent`
  ZIP contract: exact per-file SHA-256, bounded extraction, a
  credential-path safety floor, and a registry-neutral HTTPS loader. Not
  yet loadable from the UI — the contract comes first; see
  [docs/portable-agents.md](../portable-agents.md).
- Sixteen new adversarial tests attack the package reader with path
  escapes, same-length tampering, NUL and invalid UTF-8 content,
  redirect downgrades, and lying content-length headers. All rejected.
- "Add a sample spreadsheet" is now non-destructive: an existing
  workspace only gains missing sample paths; your files and the revert
  baseline are never touched.
