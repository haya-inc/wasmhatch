# v0.45.0 — Zero setup, storage safety

The landing page now sells WasmHatch's two structural advantages directly,
and the app backs the pitch with real storage controls and readable
replies.

## Highlights

- **The landing says the quiet part out loud.** A side-by-side "There is
  no setup" section compares the usual routine (install, account,
  always-on machine, config files) with opening a link, and one causal
  sentence explains why no account means both speed and safety. The
  privacy section now names where AI requests really go — only the
  provider you chose, on-device with Chrome's built-in AI.
- **Your data, pinned and portable.** A Storage panel in the chat sidebar
  shows whether the browser has agreed to keep your work, requests pinning
  in one click ("Keep my data safe"), and downloads everything as a dated
  ZIP ("Back up everything").
- **Optional key memory.** Your API key survives a reload, and a
  "Remember on this device" checkbox keeps it in this browser — only if
  you tick it. Keys are kept per provider and still go only to the
  provider you chose.
- **Replies you can read.** Assistant messages render headings, lists,
  code blocks, bold, and links as real structure instead of raw Markdown
  marks — through a dependency-free safe subset that never injects HTML
  and drops non-http(s) links.

## Under the hood

- The operator artifact index retries once when a write races it, fixing
  a flaky end-to-end test and phantom error alerts in multi-tab use.
- First end-to-end coverage for the chat surface (mocked streaming reply).
- Backup filenames use your local date.

## For contributors and forks

- CONTRIBUTING and ROADMAP now describe the chat-first direction:
  autonomy by default with visible, revertible diffs; the Business
  Operator page is a parts donor scheduled for retirement.
- New [fork guide](../fork-guide.md): fork → enable Pages → done, with
  bring-your-own Google client ID via a repository variable.
- `scripts/record-demo.mjs` records an uncut demo take against a real
  model for launch material.
